# .NET Monitoring Application

This is a .NET-based monitoring application built as part of a DevOps Bootcamp.
The application provides real-time monitoring of system resources such as memory usage and processor load through a web-based dashboard.

This project is intended for practicing build tools, application deployment, and DevOps workflows.

---

## 📌 Features

* Real-time system monitoring
* Memory usage tracking
* CPU load monitoring
* Web-based dashboard
* .NET backend
* DevOps-ready project structure
* Cloud compatible

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/dotnet-monitoring-app.git
cd dotnet-monitoring-app
```

Navigate to source folder:

```bash
cd source
```

Restore dependencies:

```bash
dotnet restore
```

---

## ▶️ Run the Application

Run the application:

```bash
dotnet run
```

Open in browser:

```text
http://localhost:5000
```

or

```text
http://localhost:8080
```

(depending on configuration)
